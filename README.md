# User Note
#### Note application with Django , ReactJS and Webpack.

# How to setup
• 	Create a virtual env for this project <br />
•	  Clone it <br />
•	  pip install -r requirements.txt <br />
•	  cd ponynote/frontend <br />
•	  npm install <br />
•	  npm run start <br />
•	  python manage.py runserver <br />
